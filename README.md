# “有恒则成” 个人博客
基于Hexo的个人博客搭建 访问地址：https://wangheng3751.github.io

## 命令：

### hexo

- 创建新文章：hexo new "title"
- 清除并生成： hexo clean && hexo g
- 本地启动：hexo s -p 端口号
- 发布：hexo d

### git

- git pull
- git add .
- git commit -m "readme"
- git push
